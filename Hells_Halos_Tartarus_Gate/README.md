# Overview
The Hells / Halos / Tartarus Gate techniques are advanced methods for dynamically searching for the SSN syscall number.

# Usage
To see the execution and the syscalls being retrieved, use the following command:
```shell
$env:RUST_LOG="debug"
cargo test
```
